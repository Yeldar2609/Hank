# Hank 🤖  
### A Custom Useless Box

Hank is a fully custom-designed useless box that automatically turns itself off whenever the switch is flipped on. The project combines CAD design, 3D printing, and simple electronics to create a responsive mechanical system.

---

## ⚙️ How It Works
When the 2-state switch is flipped ON, it activates a servo motor inside the box. The servo moves a mechanical arm that reaches out and flips the switch back OFF, completing the “useless” function.

---

## 🔌 Components
- 2-State Switch (user input)
- Servo Motor (movement)
- Limit Switch (position control & stopping point)
- 2x AAA Batteries (power source)

---

## 🧠 Design Process
All parts were designed and adjusted in CAD to ensure proper fit and function. This included:
- Resizing the enclosure to fit internal components
- Repositioning the motor for better alignment
- Testing clearances to prevent collisions
- Iterating switch designs (V1 → V3)

The final design balances functionality with simplicity while keeping the internal layout compact and adaptable.
